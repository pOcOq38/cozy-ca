# CozyCA

CozyCA is a client-side rental listing web app built with HTML, CSS, and Vanilla JavaScript.

## Features
- User authentication with session expiration
- Add / edit / delete rental listings
- Map integration with Leaflet
- Filter by location, price, beds, and baths
- User-based bookmarks

## Tech Stack
- HTML
- CSS
- JavaScript
- Leaflet
- LocalStorage

## Run Locally
Just open `index.html` in your browser.
